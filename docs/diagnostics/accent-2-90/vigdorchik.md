# «Акцент 2-90» (метод Вигдорчика)

## Общее описание

**Назначение.** Диагностика общих профессиональных навыков.

Методика М.И. Вигдорчика использует опросник из 90 вопросов для определения 11 типов акцентуаций характера.

## Инструкция

Внимательно читайте вопросы и отвечайте **ДА** [+], если согласны с данным вопросом, и **НЕТ** [-], если не согласны.
