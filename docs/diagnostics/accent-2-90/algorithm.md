# Алгоритмы расчета акцентуаций характера

## Общее описание

Данный документ описывает алгоритмы расчета результатов для трех опросников акцентуаций характера:
- **[Метод Вигдорчика](vigdorchik.md)**
- **[Метод Шмишека](schmishek.md)**
- **[Метод Шмишека-Мюллера](schmishek-muller.md)**

Все три метода используют единый алгоритм расчета, но различаются набором шкал и вопросами, которые относятся к каждой шкале.

Каждый метод описан в отдельном файле, где представлены:
- Алгоритм расчета с конфигурацией всех шкал
- Полный список вопросов опросника с указанием принадлежности к шкалам

## Структура методов

- **[Метод Вигдорчика](vigdorchik.md)** — опросник из 90 вопросов, 11 шкал акцентуаций
- **[Метод Шмишека](schmishek.md)** — опросник из 88 вопросов, 10 шкал акцентуаций
- **[Метод Шмишека-Мюллера](schmishek-muller.md)** — опросник из 90 вопросов, 11 шкал акцентуаций
